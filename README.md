# simple_sign_in
A simple Django app for handling user sign in, sign out, register, password change, and password reset operations.

[Live demo](https://www.simple-sign-in.app) (deployed on Heroku).

### Built using:

* Python 3.7
* Django 3.1.2
* Bootstrap 4
* django-crispy-forms 1.9.2
* environs 8.0.0
* googletrans 3.0.0
* boto3 1.15.16
* Gunicorn 20.0.4
* Visual Studio Code 1.50.0
* macOS 10.15.7
* Heroku
* Pipenv
